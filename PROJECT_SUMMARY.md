### Deploy to Netlify (Recommended)

1. **Create GitHub account** (free, 2 min)
2. **Push code to GitHub** (3 min)
   ```bash
   git init
   git add .
   git commit -m "Initial: Kaşifler Diyarı website"
   git remote add origin https://github.com/YOUR_USERNAME/kaşifler-website.git
   git push -u origin main
   ```
3. **Connect to Netlify** (3 min) - https://netlify.com
4. **Netlify auto-deploys** (5 min)
5. **Your site is LIVE!** ✅
