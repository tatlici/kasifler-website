This folder contains your complete, production-ready one-page website for Kaşifler Diyarî.

---

📍 START HERE:

1. Open Terminal/PowerShell in this folder
2. Run: npm install
3. Run: npm run dev
4. Open: http://localhost:3000

Your website will appear in your browser! ✨

🚀 DEPLOY TO LIVE WEBSITE:

Want to share your website with the world? Follow these steps:

1. Create GitHub account (free): github.com
2. Push this code to GitHub
3. Create Netlify account (free): netlify.com
4. Connect your GitHub repo to Netlify
5. Netlify automatically deploys!

Your website will get a free URL like: kasiflerdiyari-xyz.netlify.app

Detailed deployment guide in DEPLOYMENT.md

🎨 CUSTOMIZE:

Want to change things? Easy!

**Change text:**
- Edit `content/tr.json` (Turkish text)
- Edit `content/en.json` (English text)

**Change colors:**
- Edit `tailwind.config.ts` (update the color values)

**Add/change images:**
- Add images to `public/assets/gallery/`
- Update list in `components/Gallery.tsx`

**Change contact email:**
- Once on Netlify, set form recipient in dashboard
