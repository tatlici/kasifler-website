# Kaşifler Diyarı - Official Website

Kaşifler Diyarı is an educational game for children ages 4-12 that develops curiosity, problem-solving, and creativity.

This is the official one-page promotional website built with Next.js, React, TypeScript, and Tailwind CSS.

## Features

✨ **Responsive Design** - Mobile-first, works perfectly on all devices
🌍 **Bilingual** - Turkish (TR) and English (EN) language support
📧 **Contact Form** - Netlify Forms integration with spam protection
🎬 **Video Hero** - Full-width video background with fallback
🖼️ **Lazy Loading** - Optimized gallery with intersection observer
♿ **Accessible** - WCAG 2.1 AA compliance with semantic HTML
⚡ **Performance** - Optimized images, CSS-in-JS, minified assets
🎨 **Modern Styling** - Tailwind CSS with custom theme colors

## Tech Stack

- **Framework**: Next.js 14+ with App Router
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Hosting**: Netlify
- **Forms**: Netlify Forms
- **Images**: Next.js Image component with WebP optimization
- **Animations**: CSS animations with Intersection Observer

## Installation

```bash
# Clone or download the project
cd kaşifler-website

# Install dependencies
npm install

# Run development server
npm run dev

# Open http://localhost:3000
```

## Building for Production

```bash
npm run build
npm start
```

## Deployment to Netlify

1. Push your code to GitHub
2. Connect to Netlify from GitHub
3. Netlify will automatically detect Next.js and deploy
4. Configure domain settings in Netlify dashboard

## Language Switching

The site supports Turkish (TR) and English (EN). Language preference is saved in localStorage and persists across sessions.

## Contact Form

The contact form uses Netlify Forms for handling submissions. Submissions are sent to `info@kasiflerdiyari.com` by default. Configure by updating the email in Netlify dashboard.

## Common Tasks

### Adding a New Section

1. Create component in `components/`
2. Add content to `content/tr.json` and `content/en.json`
3. Import and add to `app/page.tsx`

### Updating Translations

Edit `content/tr.json` for Turkish and `content/en.json` for English.

### Customizing Colors

Edit `tailwind.config.ts` theme colors:
- `primary`: #FF6B45 (orange)
- `secondary`: #4ECDC4 (teal)
- `accent`: #FFE66D (yellow)
- `dark`: #2C3E50 (dark blue)

### Adding Images

Place images in `public/assets/` and reference with `/assets/...` path.

## Troubleshooting

**Form not working?**
- Ensure form `name="contact"` attribute is set
- Check Netlify dashboard for submissions
- Verify email configuration in Netlify

**Videos not playing?**
- Check video file path: `public/assets/hero/video.mp4`
- Ensure MP4 codec compatibility
- Add poster image for better UX

**Images not loading?**
- Verify image paths in gallery folder
- Check image file permissions
- Ensure images are in correct format
